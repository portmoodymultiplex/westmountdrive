The History of This File

Four Years. Four Applications. One Property. One Family.

Same property. Same goal. Same four families. Everything the City asked for changed completely.

2021 — Application 1
RS-1S bare land strata subdivision. Front-to-back lot split. 4 dwellings. City instruction at the time: daylight the ditch along Grouse Lane. All parking accessed from Westmount Drive at the front. Front access was not only permitted — it was the direction staff gave.
2021–2023
Application stalls. City begins internal work on a "front-and-back zone" concept intended to address exactly this type of lot. That concept is never enacted. The applicant waits.
November 2023
Bill 44 receives Royal Assent. SSMUH mandated. Applicant pivots to the new framework. Same property. Same goal. New process.
June 2024
Port Moody updates its zoning bylaw to comply with Bill 44 SSMUH requirements.
July 2, 2025 — Application 4
SSMUH development permit application submitted. Architrix Design Studio. File MDP000108. 4 dwellings. Same property. Over $100,000 already spent across prior applications.
October 8, 2025
Staff comments letter. Instruction: all parking must now be accessed from Grouse Lane at the rear. Front access not permitted. Ditch: no longer required to be daylighted — now just bridged with a culvert. Environmental concern that justified daylighting in 2021 is not addressed.
March 30, 2026 — Today
Still no permit. Access decision unresolved. BC Hydro queue at risk. Same applicant. Same property. Same four families.
The Argument the City Will Make — and the Response

Staff will note that the 2021 application was a subdivision under RS-1S and the 2025 application is an SSMUH development permit — different application types with different regulatory frameworks. That is correct as far as it goes.

It does not go far enough. In both cases: four dwellings, four families, same lot, same physical site conditions. The application type changed. The land did not. The grade did not. The ditch did not. What changed is what the City required — and in several cases those requirements directly contradict each other.
